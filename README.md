Disaster Recovery System on Cloud


Overview

This project demonstrates a simple disaster recovery system hosted on the cloud.

The application can:
- View current data
- Create backups
- Simulate data loss
- Restore data from backup

Technologies Used

- Python
- Flask
- Render Cloud

API Endpoints

GET /
GET /data
POST /backup
POST /simulate-disaster
POST /restore

Live Deployment

Hosted on Render Cloud.

Conclusion

This project demonstrates the basic concept of backup and recovery in cloud environments.
